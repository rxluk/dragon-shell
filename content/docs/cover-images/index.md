## Add cover image

```yaml
---
title: "your-post-title"
# ...
cover:
    image: example.jpg
    alt: xxx # Optional
---
```

### For author, add avatar image

```yaml
---
title: "author-name"
# ...
avatar:
    image: example-avatar.jpg
    alt: Example Avatar
cover:
    image: example-background.jpg
    alt: Example Background Image
```
